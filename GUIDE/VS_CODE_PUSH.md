## 🚀 Push from VS Code

### Method 1: Using VS Code Git Panel

1. **Open Source Control** (Ctrl+Shift+G)
2. Click the **"..." menu** (three dots) at the top
3. Select **"Push"** or **"Push to..."**
4. Choose **"origin"** as remote
5. Branch: **main**

### Method 2: Using VS Code Command Palette

1. Press **Ctrl+Shift+P**
2. Type: `Git: Push`
3. Select **"origin"**
4. Branch: **main**

### Method 3: Using Terminal in VS Code

Open VS Code terminal (Ctrl+`) and run:

```powershell
git push -u origin main
```
